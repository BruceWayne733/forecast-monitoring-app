Forecast Monitoring App

UK wind forecast accuracy, with the horizon logic exposed.

Compare hourly wind generation actuals against the latest eligible national forecast for the selected period. Forecast eligibility means the forecast must have been published at least the chosen number of hours before the target time.

Data sources

BMRS FUELHH + WINDFOR

Actuals are normalized to hourly averages so they line up with the forecast timestamps.
